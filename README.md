# Screentime

Measure your screentime and appusage on bspwm. Logs daily.



## Usage

Start daemon:
```sh
screentime.sh subscribe
```

Print today's app usage:
```sh
screentime.sh
# or
screentime.sh show
```

For more information see:
```sh
screentime.sh help
```
